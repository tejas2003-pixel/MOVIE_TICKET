About Movie Magic - Ticket Booking System

Movie Magic is an innovative and user-friendly movie ticket booking system designed to simplify the process of reserving seats for your favorite movies. This system enables users to browse movies, check show timings, select preferred seats, and complete bookings seamlessly online.

Project Objectives

  • Provide a smart and interactive movie ticket booking experience.
  • Allow users to register, log in, and manage their bookings.
  • Enable real-time seat selection with visual seating layouts.
  • Facilitate secure and multiple payment options for bookings.
  • Offer movie recommendations based on user preferences.

Key Features

  • Simple registration and login system for users.
  • Dashboard for browsing currently available movies.
  • Detailed movie info including ratings, showtimes, and trailers.
  • Dynamic seat booking with availability updates.
  • Confirmation ticket display and download option.
  • Contact and support information for users.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python with Flask framework
  • Database: SQLite
  • Styling & UX: Modern CSS with gradients and animations
